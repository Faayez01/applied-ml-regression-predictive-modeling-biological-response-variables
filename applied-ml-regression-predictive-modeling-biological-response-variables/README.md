# Applied ML Regression for Predictive Modeling of Biological Response Variables

Interpretable machine learning foundation for continuous biological outcome prediction.

![Project figure](assets/figure-1.png)

## Overview

Builds and interprets linear regression models for biological datasets. The analysis models plant species richness as a function of added nutrients and predicts mutation counts from father age using slopes, standard errors, R-squared, and hypothesis tests.

This is the strongest resume project in the set because it connects directly to supervised machine learning concepts: a quantitative predictor is used to estimate a continuous response, evaluate explained variance, and generate predictions.

## Tools and Methods

- Linear regression

- Supervised regression framing
- Model coefficient interpretation
- Explained variance
- Statistical significance testing

## Key Work

- Modeled how nutrient addition relates to plant species richness.
- Estimated the relationship between father age and number of new mutations.
- Used regression output for prediction and biological explanation.
- Interpreted slope coefficients as expected response change per unit increase in a predictor.
- Used R-squared to quantify how much variation the model explained.
- Generated predictions from the fitted regression equation.

## Data Science Framing

- **Problem type:** Supervised regression for continuous biological outcomes.
- **Features:** Nutrient addition count and father age.
- **Response variables:** Plant species richness and number of new mutations.
- **Model:** Interpretable baseline linear regression.
- **ML extension:** This could be expanded with train/test validation, residual diagnostics, regularized regression, or comparison against nonlinear models such as random forests.

## Dataset

Grassland nutrients and father-age mutation datasets from Whitlock and Schluter textbook resources.
